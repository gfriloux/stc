## Qu'est-ce que le STC ?

Dans l'univers de Warhammer 40 000, les **Standard Template Constructs** sont d'anciennes
bases de données contenant l'intégralité du savoir technologique de l'Humanité — des plans
si sacrés que la découverte du moindre fragment est célébrée à travers des secteurs entiers.

Ce projet emprunte ce nom, et l'ambition derrière.

**STC est une bibliothèque de flakes Nix** — une collection soigneusement maintenue de
modules NixOS et Home Manager réutilisables, que des flakes externes consomment pour
construire leurs systèmes. STC ne contient aucune configuration système en propre.
C'est un outillage, rien de plus.

Ton flake de production déclare le système. STC fournit les briques.
