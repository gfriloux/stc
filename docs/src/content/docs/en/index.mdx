## What is the STC?

In the lore of Warhammer 40,000, **Standard Template Constructs** are ancient databases
containing the complete technological knowledge of Humanity — blueprints so sacred that
discovering even a fragment is cause for celebration across an entire sector.

This project borrows that name, and the ambition behind it.

**STC is a Nix flake library** — a curated collection of reusable NixOS and Home Manager
modules that external flakes consume to build their systems. STC contains no system
configurations of its own. It is purely a toolkit.

Your production flake declares the system. STC provides the building blocks.
